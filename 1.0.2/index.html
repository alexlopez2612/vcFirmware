MODO DE OPERACIÓN

Si el valor actual de AI1 está dentro de un rango activo (incluyendo sus límites), AO1 toma el valor configurado. Las filas no pueden solaparse, pero puede haber huecos. Si ningún rango coincide, la salida no se modifica.

Enable Min AI1 Max AI1 Salida AO1

En el modo Auto 3, la salida AO1 intenta seguir el valor de la entrada AI1.

• Si la diferencia es menor que el Deadband, el controlador no actúa.
• Si la diferencia está entre Deadband y Db para regular, la salida se ajusta directamente al valor de la entrada.
• Si la diferencia es mayor que Db para regular, la salida se mueve poco a poco en pasos de Incremento / Decremento, esperando el tiempo configurado en T. Inc / Dec entre cada paso.

Después de modificar la salida, el sistema espera el tiempo de T. Estabilización antes de volver a actuar. Si no se ha realizado ningún cambio, esperará el tiempo de T. entre lecturas para comprobar de nuevo la diferencia.

IMPORTANTE: AI1 y AO1 deben estar configuradas en la misma unidad.

Programa hasta 12 franjas por día. Las franjas OFF no actúan pero conservan su configuración.
La salida analógica se aplica cuando la hora actual está dentro de una franja activa.
El inicio de la franja está incluido y el final no (Ejemplo: 08:00–10:00 actúa hasta las 09:59).
Las franjas no pueden solaparse ni cruzar de un día a otro.
Si no hay ninguna franja activa, la salida no se modifica.

Franja Enable Inicio Fin Valor AO1

Programa hasta 12 franjas por día para actuar sobre los relés.
Cada franja puede ordenar una acción sobre un relé: OFF, ON o PULSE.
En PULSE, el relé solo se activa al entrar en la franja y se mantiene el tiempo global configurado.
El inicio de la franja está incluido y el final no (Ejemplo: 08:00–10:00 actúa hasta las 09:59).
Las franjas no pueden solaparse ni cruzar de un día a otro.
Si no hay ninguna franja activa, el controlador no modifica los relés.
